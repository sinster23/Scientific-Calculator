3. For MODE,4 options are provided Select any option with     numbers provided for each.
    For Quadratic type equation in 'ax^2+bx+c' format and hit     MODE for result.
    For Derivatives type the question & hit MODE for result.
    For Determinants syntax is 'det([a,b,c];[d,e,f])' then hit MODE     for result.
    For matrix only (2x2)matrix operations are available(+,-,*)     eg:    [[2,3]*[5,6]] hit MODE for result.
